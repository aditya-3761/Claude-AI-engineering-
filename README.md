I enjoy building full-stack web applications using modern technologies such as React, Node.js, MongoDB, and Python. Alongside development, I am continuously exploring AI tools and learning how to leverage Large Language Models (LLMs) effectively.

## 🎯 My Goals for the Next 60 Days

By the end of this challenge, I aim to:

* Learn the fundamentals of Prompt Engineering.
* Understand how Large Language Models (LLMs) process prompts.
* Master techniques such as:

  * Zero-Shot Prompting
  * Few-Shot Prompting
  * Chain-of-Thought Prompting
  * Role-Based Prompting
  * Structured Output Prompting
  * Prompt Chaining
* Build real-world AI use cases using prompt engineering.
* Improve AI-generated outputs through prompt optimization.
* Document daily learnings and experiments.
* Create a portfolio of prompt engineering projects.
* Develop practical skills that can be applied in AI, software development, and automation.
